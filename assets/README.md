# Assets

Düstur sisteminin görsel kaynak kütüphanesi.

## Motifler (`motifs/`)

Selçuklu motif kütüphanesi. Tüm motifler SVG · vector · ölçeklendirilebilir.

| Dosya | Kullanım |
|---|---|
| [`selcuklu-rosette.svg`](./motifs/selcuklu-rosette.svg) | Sayfa marjlarında düşük-opaklık arka plan (Bölüm I · 1.3) |
| [`sigil.svg`](./motifs/sigil.svg) | Anayasa kademesi mührü · TBK rengi |
| [`elmas-zinciri.svg`](./motifs/elmas-zinciri.svg) | Bölüm ayraç · Lacivert renk |

## Kullanım Kuralı

Bezeme değil **imza** amacıyla kullanılır. Hiçbir motif metin akışını bozacak yoğunlukta yerleşmez.

| Yerleşim | Opacity | Boyut |
|---|---|---|
| Sayfa marjı arka plan | 0.05-0.08 | ~200mm |
| Bölüm açılışı sigil | 0.05 | ~100mm |
| Bölüm ayraç (zinciri) | 0.08-0.15 | tam genişlik |
| Mühür (rozet) | 1.0 | inline (5-10mm) |
